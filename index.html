Computer Vision · Edge AI 2024 — 2025
Insulator Defect Detection System

Power transmission lines need regular inspection, but sending people up towers is dangerous and expensive. This project uses a custom-designed Ghost-Hybrid YOLO11 model to detect damaged insulators from drone/camera footage — and it runs directly on affordable edge hardware like Raspberry Pi and NVIDIA Jetson. Exported to ONNX, TensorRT, and TFLite for flexible deployment.

LLM Fine-Tuning · NLP Jul — Sep 2025
MedChat: Medical AI Assistant

Fine-tuned Mistral 7B on real medical dialogues using QLoRA — making it possible to train a 7-billion parameter model on consumer hardware with 4-bit quantization. Includes a Human-in-the-Loop pipeline: the model flags its own uncertain responses for doctor review, then learns from that feedback. Dynamic few-shot prompting makes responses feel more empathetic and contextually appropriate.
